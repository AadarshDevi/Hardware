# NVIDIA GeForce RTX Graphics Cards

## Example Cards
1. 3060
2. 4080 Super
3. 5070Ti

## Naming Convention
1. First 2 digits (30, 40, 50, etc) is the generation of the GPU. Higher the number, the more expensive it is. each generation usually have an improved architecture.
2. Last 2 digits (50, 60, 70, 80, 90) show how powerful the gpu is. higher the number, the more powerful and expensive it is.
3. Ti/Super - Upgraded/Refreshed versions of the base model.
